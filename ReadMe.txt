Предложение по модернизации линии - "внедрение программы отслеживания дефектов"
Исходная лента/прокат
Задача: отследить, зафиксировать наличие дефектов исходной ленты/проката во время задачи в линию.  
Результат: выдача файла с отметкой по наличию дефекта с описанием: вид, расположение, интервал

Готовая продукция:
Задача: отследить, зафиксировать наличие дефектов готовая продукция во время прохождения участка выходная секция - сопоставление с данными входная секция
Результат: выдача файла с отметкой по наличию дефекта с описанием: вид, расположение, интервал, сопоставление с исходной лентой
Программа должна быть "самообучающаяся" с возможностью корректировки данных

В дальнейшем внедрение данных по критичности дефектов

Итог: 
- упрощение претензионной работы по доказательству дефектов полимерного покрытия отображающихся с исходной ленты
- экономия времени по решению выбора статуса готовой продукции (отсутствие необходимости распаковки рулонного проката готовой продукции на осмотр критичности дефектов)

Начинать предлагаю с простого и явного и частого наддавы/наколы. Это дефекты возможны как с исходного проката/ленты так и образоваться во время окрашивания.
